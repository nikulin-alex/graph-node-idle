# Graph Idle Simulator

**Строй графы, улучшай узлы, наблюдай за обходчиками.**

Graph Idle Simulator — это idle-игра (инкрементальная), в которой вы создаёте и развиваете граф из узлов. Узлы приносят пассивный доход, а по рёбрам графа бегают разноцветные обходчики, использующие алгоритмы BFS и DFS. Чем выше уровень узла — тем больше он приносит.

---

## 🎮 Геймплей

Всё начинается с одного узла. Кликайте на него, чтобы открыть панель управления, и:

- **➕ Добавляйте новые узлы** — каждый новый узел соединяется с текущим и начинает приносить доход.
- **⬆️ Улучшайте узлы** — повышайте уровень узла, его доход растёт квадратично (доход = уровень²).
- **🗑️ Удаляйте узлы** — за 100 монет можно удалить лишний узел (кроме самого первого).

Доход начисляется **каждую секунду** автоматически. Чем больше узлов и чем они выше уровнем — тем быстрее растёт ваш баланс.

### 🤖 Обходчики графа (Traversers)

Это ключевая механика игры. Обходчики — это агенты, которые путешествуют по вашему графу, используя алгоритмы обхода:

- **BFS (Breadth-First Search)** — обход в ширину, исследует граф по уровням.
- **DFS (Depth-First Search)** — обход в глубину, уходит вглубь по цепочкам узлов.

Каждый обходчик приносит дополнительный доход, равный доходу узла, на котором он находится в данный момент. У обходчиков есть множитель дохода, и они могут быть разных цветов.

Обходчиков можно покупать в **Магазине обходчиков** (кнопка в правом верхнем углу экрана). Цена растёт с каждым купленным обходчиком.

## 🖱️ Управление

| Действие | Описание |
|---|---|
| **ЛКМ по узлу** | Выделить узел, открыть панель действий |
| **Перетаскивание узла** | Зажать ЛКМ на узле и переместить |
| **Перетаскивание фона** | Зажать ЛКМ на пустом месте — перемещение камеры |
| **Колёсико мыши** | Приблизить / отдалить (зум) |
| **Кнопка «Магазин обходч.»** | Открыть магазин обходчиков (справа вверху) |

## 🚀 Как запустить

### Зависимости

- Python 3.10+
- [pygame-ce](https://github.com/pygame-community/pygame-ce) 2.5.7 (Community Edition)

Установка зависимостей:

```bash
pip install -r requirements.txt
```

### Запуск

```bash
python main.py
```

Игра запускается в окне, размер которого составляет 90% от разрешения вашего экрана. Окно можно изменять размером (RESIZABLE).

## ⚙️ Технические детали (кратко)

- **Доход узла** = `уровень²` монет в секунду.
- **Цена улучшения** = `10 × 1.5^уровень`.
- **Цена нового соединения** рассчитывается через сигмоиду: чем больше связей у узла, тем дороже (до 5000).
- **Максимум соединений узла**: 10 (5 «ёмкость» × 2).
- **Максимальный уровень узла**: 20 (10 «ёмкость» × 2).
- **Цена удаления узла**: 100 монет (первый узел удалить нельзя).
- **Базовая цена обходчика**: 500, растёт с коэффициентом 1.8 за каждого купленного.
- **Обходчики** используют BFS для поиска кратчайшего пути между узлами и перемещаются по нему шаг за шагом.
